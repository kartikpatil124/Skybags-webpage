# Skybags Webpage

A simple static webpage for Skybags, showcasing backpacks and luggage products. Built with HTML, CSS, and JavaScript.

## Files
- `index.html`: Main homepage
- `backpack.html`: Backpack section
- `luggage.html`: Luggage section
- `script.js`: JavaScript for interactivity
- `sky1.css`: Stylesheet
- `img/`: Image assets

## Usage
Open `index.html` in a web browser to view the site.
